
## 스택(Stack)
- **정의**: LIFO(Last-In, First-Out) 구조. 마지막에 넣은 데이터가 가장 먼저 나옴.
- **주요 함수/메서드**:
	- `push(value)`: 값 추가 (맨 뒤)
	- `pop()`: 값 제거 및 반환 (맨 뒤)
	- `peek`(JS에는 없음, 대신 `stack[stack.length-1]`): top 값 확인
	- `length`: 현재 스택 크기
```js
let stack = [];
stack.push(1); // [1]
stack.push(2); // [1,2]
stack.pop();   // 2, stack: [1]
let top = stack[stack.length-1]; // 1
```

- **코딩테스트 활용 예시**:
	- 괄호 짝 검사, 후위/중위/전위 표기식 변환 및 계산
	- DFS(깊이 우선 탐색) 구현
	- 최근 사용 데이터 저장(LRU)
	- 문자열 뒤집기, 연속된 데이터 처리

---

## 큐(Queue)
- **정의**: FIFO(First-In, First-Out) 구조. 먼저 넣은 데이터가 먼저 나옴.
- **주요 함수/메서드**:
	- `push(value)`: 값 추가 (맨 뒤)
	- `shift()`: 값 제거 및 반환 (맨 앞)
	- `length`: 현재 큐 크기
```js
let queue = [];
queue.push(1); // [1]
queue.push(2); // [1,2]
queue.shift(); // 1, queue: [2]
```

- **코딩테스트 활용 예시**:
	- BFS(너비 우선 탐색) 구현
	- 시뮬레이션(공주 구하기, 카드 게임 등)
	- 순서대로 처리해야 하는 데이터(대기열, 이벤트 등)
	- 슬라이딩 윈도우, 라운드 로빈, 회전 큐

---

## 스택/큐를 활용해야 하는 코딩테스트 유형
- 괄호 짝 검사, 수식 계산(후위/중위/전위)
- 그래프 탐색(DFS: 스택, BFS: 큐)
- 시뮬레이션(순서대로 처리, 제거, 회전)
- 최근 데이터 관리(LRU, 히스토리)
- 문자열/배열의 연속성, 구간 처리

**팁:**
- JS에서 큐는 `push`+`shift`, 스택은 `push`+`pop`으로 구현
- 큐의 성능이 중요할 때는 `LinkedList`(JS엔 없음)이나 `deque` 라이브러리 사용 가능
